算法-有环链表的数学解法

Posted on 2024-01-14

判断链表中是否有环

描述

判断给定的链表中是否有环。如果有环则返回true，否则返回false。

思路

双指针

使用两个指针，fast 与 slow。它们起始都位于链表的头部。随后，slow 指针每次向后移动一个位置，而fast 指针向后移动两个位置。如果链表中存在环，则 fast 指针最终将再次与 slow 指针在环中相遇。

图片引用路径

1. 本地图片渲染

在 站点配置文件 中设置post_asset_folder: true，当该配置被应用后，使用hexo new命令创建新文章时，会生成相同名字的文件夹，也就是文章资源文件夹。

若文件名为postname.md，则创建新博客时会在同一级目录下创建postname文件夹，用于保存图片

使用Typora时，设置图片文件保存路径: ./${filename} 即保存到与 当前正在编辑的文件名 相同的同级文件夹下，和Hexo的要求一致。

现在，我们在 Typora下使用 ![example](postname/example.jpg)引用图片，享受实时预览，

2. 转换引用路径: hexo-asset-img

但是Hexo渲染时，不会为图片引用补上前缀，即为\example.png而非\webpath\example.png

使之发布后能正确加载我们的图片，还需要做以下转换:![example](postname/example.jpg) --> {% asset_img example.jpg example %}

使用npm install hexo-asset-img --save引入hexo-asset-img
